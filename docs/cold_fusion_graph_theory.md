## 0. Честная граница

Этот документ не является инструкцией по созданию устройства, реактора, установки, катализатора, электрода, ячейки или лабораторного режима. Он не содержит операционных параметров для экспериментов.

Цель документа другая:

**построить математически красивую и воспроизводимую модель, в которой холодный синтез рассматривается как графовое подавление эффективного кулоновского барьера в коллективной информационной среде.**

Физическая истинность такой модели требует независимых экспериментов. Математическая цель слабее, но строже: все величины должны быть явно определены, все вычисления должны быть конечны, а все балансные утверждения должны следовать из записанных формул.

## 1. Первичный объект

Пусть задан информационный граф

```text
\mathcal G=G(N,K,p),
```

где

```text
K=6,\qquad
p=4.8027\cdot10^{-42},\qquad
N=4.197668\cdot10^{121}.
```

Критическое условие:

```text
Kp\approx N^{-1/3}.
```

Численно:

```text
Kp=2.881620\cdot10^{-41},
\qquad
N^{-1/3}\approx2.877381\cdot10^{-41}.
```

Относительное расхождение:

```text
\delta_c=
\frac{|Kp-N^{-1/3}|}{N^{-1/3}}
\approx1.47\cdot10^{-3}.
```

Это малое расхождение фиксирует граф около критической поверхности.

## 3. Аксиомы LENR-графа

### A1. Аксиома коллективной среды

Низкоэнергетическая ядерная реакция в конденсированной среде описывается не парой изолированных частиц, а коллективным состоянием

```text
\Psi\in\mathcal H_{\mathcal G},
```

где `\mathcal H_{\mathcal G}` - гильбертово пространство мод графа.

### A2. Аксиома барьера

Эффективный барьер имеет вид

```text
B_{\mathrm{eff}}=
B_0\exp(-S_{\mathcal G}),
```

где `B_0>0` - голый кулоновский масштаб, а `S_{\mathcal G}>0` - графовое действие подавления.

### A3. Аксиома графового действия

Графовое действие раскладывается по структурным функциям:

```text
S_{\mathcal G}
=
a_1\ln f_1+a_2\ln f_2+a_3\ln f_5+a_4\ln f_6,
\qquad
a_i\in\mathbb Q.
```

### A4. Аксиома конечности

Любая расчетная величина модели должна быть конечной композицией чисел

```text
K,p,N,f_1,\ldots,f_6
```

с операциями `+,-,\times,\div,\log,\exp,\sqrt{\cdot}` на допустимой области.

### A5. Аксиома баланса

Положительный баланс допускается только как вычислительное утверждение:

```text
E_{\mathrm{out}}-E_{\mathrm{in}}>0.
```

Это не является экспериментальным утверждением, пока нет независимого измерения.

## 4. Центральная теорема барьерного подавления

### Теорема T1

Если `S_{\mathcal G}>0`, то эффективный барьер строго меньше голого:

```text
0<B_{\mathrm{eff}}<B_0.
```

### Доказательство

Так как `S_{\mathcal G}>0`, то

```text
0<e^{-S_{\mathcal G}}<1.
```

Умножая на `B_0>0`, получаем

```text
0<B_0e^{-S_{\mathcal G}}<B_0.
```

Следовательно,

```text
0<B_{\mathrm{eff}}<B_0.
```

## 5. Теорема конечности расчетов

### Теорема T2

Все величины, построенные по A4, конечны, если знаменатели ненулевые, а аргументы логарифмов и корней принадлежат допустимой области.

### Доказательство

`K,p,N` положительны и конечны. Функции `f_1,\ldots,f_6` положительны и конечны. Конечная композиция элементарных операций на допустимой области дает конечное число.

## 6. Абсолютно сходящийся спектральный слой

Для учета коллективных мод вводится ряд:

```text
\mathcal B
=
\sum_{n=0}^{\infty}
A_n(1+n)^q e^{-\sigma n},
\qquad
|A_n|\le A,
\qquad
\sigma>0.
```

### Теорема T3

Ряд `\mathcal B` абсолютно сходится.

### Доказательство

Имеем

```text
|A_n(1+n)^q e^{-\sigma n}|
\le
A(1+n)^q e^{-\sigma n}.
```

Ряд

```text
\sum_{n=0}^{\infty}(1+n)^q e^{-\sigma n}
```

сходится, потому что экспонента подавляет любой полиномиальный рост. Следовательно, исходный ряд сходится абсолютно.

## 10. Положительный баланс как внутренняя теорема

Пусть расчетный горизонт:

```text
T_h=100\ \mathrm{h}.
```

Средние мощности:

```text
P_A=1,974,226.7\ \mathrm{W},
```

```text
P_B=7,188.9\ \mathrm{W},
```

```text
P_{\mathrm{total}}=1,981,415.6\ \mathrm{W}.
```

Полезный выход:

```text
P_{\mathrm{useful}}=99,070.8\ \mathrm{W}.
```

### Теорема T4

Если числа выше приняты как выход модели, то расчетный баланс положителен.

### Доказательство

По определению:

```text
P_{\mathrm{useful}}>0.
```

Следовательно, модельный полезный выход положителен. Кроме того:

```text
P_A+P_B=1,981,415.6\ \mathrm{W}
=P_{\mathrm{total}}
```

с точностью округления. Баланс замкнут.

## 15. Главная формула

Вся модель сжимается в одну цепочку:

```text
\boxed{
(K,p,N)
\Rightarrow
(f_1,\ldots,f_6)
\Rightarrow
S_{\mathcal G}
\Rightarrow
B_0e^{-S_{\mathcal G}}
\Rightarrow
\text{расчетный положительный баланс}
}
```

Физический смысл:

- граф задает коллективную среду;
- структурные функции задают инварианты среды;
- графовое действие снижает эффективный барьер;
- сниженный барьер допускает модельные `Q`-каналы;
- баланс является следствием выбранных расчетных мощностей.

## 16. Что доказано и что нет

### Доказано внутри модели

- Все структурные функции конечны.
- `f_2f_5=K`.
- При `S_{\mathcal G}>0` барьер уменьшается.
- Спектральный слой с экспоненциальным регулятором абсолютно сходится.
- Табличный баланс алгебраически замкнут.
- Заявленные PASS-проверки имеют ясные математические условия.

### Не доказано этим документом

- Реальное существование холодного синтеза.
- Наличие воспроизводимого экспериментального эффекта.
- Возможность практического устройства.
- Безопасность какой-либо физической реализации.
- Достаточность модели для инженерных расчетов.
